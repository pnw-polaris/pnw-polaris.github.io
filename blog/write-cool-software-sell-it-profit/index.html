The Shareware starter kit is something Michael Lehman, myself and Dan Fernandez partnered on this year. Right now there are Visual Basic and C# versions, and a C++ version is on its way.

The Starter Kit is a set of classes that you can easily incorporate into your .NET Framework 2.0 application that gives you a splash screen, payment via PayPal, Product Activation, Product Registration, Event/Error Reporting, and Product Feedback.
